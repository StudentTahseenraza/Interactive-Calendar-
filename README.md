## 📖 Overview

> 🧠 **Designed for clarity. Built for productivity.**

Wall Calendar transforms the traditional calendar into a **modern web experience**.  
With stunning visuals, intuitive interactions, and flexible note-taking, it helps you:

- 📅 Plan events efficiently  
- 🧩 Organize tasks visually  
- 🚀 Boost daily productivity  

---

<div align="center">

| 🚀 Feature | 💡 Description |
|-----------|--------------|
| 🖼️ **Monthly Hero Images** | Beautiful, dynamic visuals that change every month |

## 📦 Installation

### ⚙️ Prerequisites

- Node.js (v18+)
- npm / yarn / pnpm

---

### 🚀 Setup Steps

    ```bash
    # 1️⃣ Clone the repository
    git clone https://github.com/yourusername/wall-calendar.git
    
    # 2️⃣ Navigate into the project
    cd wall-calendar
    
    # 3️⃣ Install dependencies
    npm install
    
    # 4️⃣ Start development server
    npm run dev

🎮 Usage Guide
📅 Calendar Navigation

| Action         | Method                              |
| -------------- | ----------------------------------- |
| Previous Month | Click ← button or press ← arrow key |
| Next Month     | Click → button or press → arrow key |
| Today          | Click **Today** button              |
| Jump to Month  | Click on mini calendar preview      |

🎯 Date Selection

| Action             | Method                                 |
| ------------------ | -------------------------------------- |
| Select Single Date | Click any day                          |
| Select Date Range  | Click start date → then click end date |
| Clear Selection    | Click ❌ next to selected range         |

Actions:

| Action     | Method                                      |
| ---------- | ------------------------------------------- |
| ✏️ Edit    | Click edit icon on the note                 |
| 🗑️ Delete | Click trash icon                            |
| ➕ Add      | Type note + press Enter / click Plus button |
